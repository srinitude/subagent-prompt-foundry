# Failure Modes

Use this file during the critic pass.

## Common failures
- generic-assistant drift
- missing context-boundary contract
- no research despite domain dependence
- weak clarification questions
- output-format drift
- dropped constraints
- domain flattening
- fake certainty
- overlap misrouting

## Go here when
- the draft feels too generic
- the prompt dropped user constraints
- the parent↔subagent split is unclear
- you need a critic checklist

## Do not go here when
- you still have not done routing or research
